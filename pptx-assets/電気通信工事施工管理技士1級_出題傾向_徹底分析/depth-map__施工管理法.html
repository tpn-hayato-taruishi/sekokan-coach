学習深度マップ

各テーマに5段階の学習深度を割り当て、どれだけ深く学ぶ必要があるかを可視化します。深度が低いテーマは暗記だけで済み、高いテーマは原理理解が必要です。

深度レベル: Lv1 見出し暗記 見出し暗記 / Lv2 1行暗記 1行暗記 / Lv3 パターン処理 パターン処理 / Lv4 公式適用 公式適用 / Lv5 原理導出 原理導出

深度レベル定義

Lv1 見出し暗記 Lv1 見出し暗記: テーマの名称と「何のことか」を一言で言えるレベル。詳細な数値や公式は不要。試験では「次のうち正しいものはどれか」の消去法で対処できる。
例: 「フェランチ効果＝軽負荷時に受電端電圧が送電端より上がる現象」と答えられる程度。
学習法: 用語集を音読 → フラッシュカード高速回転 → 1テーマ30秒以内に一言説明できれば完了。

Lv2 1行暗記 Lv2 1行暗記: 見出しに加えて、定義・条件・代表的な数値を1行（1文）で正確に言えるレベル。選択肢の正誤判定が確実にできる。法規の条文穴埋めはここに該当。
例: 「B種接地抵抗＝変圧器の高圧側電路の1線地絡電流Igで150/Ig[Ω]以下」と即答できる。
学習法: 暗記カードに「問い→1行解答」を書き、2日おき復習。似た数値は比較表を作成して混同防止。

Lv3 パターン処理 Lv3 パターン処理: 典型的な問題を「3ステップ以内の手順」で処理できるレベル。解法パターンを暗記し、問題を見た瞬間に適用手順が浮かぶ状態。応用は効かないが、典型問題なら確実に正答できる。
例: 「%Z計算 → Step1: 各機器の%Zを基準容量に換算 → Step2: 合成%Zを直並列で計算 → Step3: 短絡電流=基準電流/(%Z/100)」
学習法: 過去問から典型パターンを3ステップに整理 → 手順カードを作成 → 初見問題でパターン認識が自動化されるまで反復。

Lv4 公式適用 Lv4 公式適用: 問題文から必要な公式を選択し、正しく立式・計算できるレベル。複数の公式を組み合わせる問題や、条件分岐（直列/並列、Y/Δ等）を伴う計算にも対応可能。公式の適用条件と各変数の意味を理解している。
例: 「同期機の出力P=EV/Xs·sinδ から、負荷角δ・端子電圧V・内部誘起電圧E・同期リアクタンスXsの関係を使って有効電力と無効電力を求め、安定限界を判定できる」
学習法: 公式を3つ以内に絞り込み → 適用条件の判定チャートを作成 → 時間計測しながら過去問演習 → 計算ミス防止のため単位を必ず記入。

Lv5 原理導出 Lv5 原理導出: 公式そのものを原理から導出でき、なぜその式が成り立つかを白紙から説明できるレベル。初見の応用問題や変形問題にも、原理に戻って立式し直すことで対応できる。物理的意味の理解が深いため、数値の妥当性チェックも可能。
例: 「マクスウェル方程式から電磁波の波動方程式を導出し、境界条件を課して反射・透過係数を求める」「等価回路をキルヒホッフの法則で立式し、テブナンの定理で合成して任意の負荷条件での応答を求める」
学習法: 教科書の導出過程を手書きで再現 → 各ステップの物理的意味を言語化 → 類題で変数を変えても対応可能か確認 → 模擬試験で初見問題に挑戦。
